# faq
As a user I should get a notification when a question is being answered and when an answer is modified or edited. I should get an email when any of the above things happens.

 

Acceptance Criteria

路        The user should get an email when a question is being answered. The email should be addressed to the registered users.

路        The email should have a subject, briefly line explaining the purpose of the email.

路        The body should have main points and a link to the question being answered.

路        The email should end with a greeting.
